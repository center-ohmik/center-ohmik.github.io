### vi 문자열 치환(바꾸기) 명령어
```
:%s/찾는 문자열/바꿀 문자열
```

### 예제로 사용한 명령어(정규표현식 사용 가능)
```
:%s/^/-A INPUT -s /g        //모든 문자열 제일 앞 부분에 '-A INPUT -s ' 추가
:%s/$/ -j DROP/g            //모든 문자열 제일 뒷 부분에 ' -j DROP' 추가
```
